# BASE TYPESCRIPT

### DESCRIPTION
Base microservices typeScript and SOLID

### INIT YARN
```
yarn init -y 
```
### ADD LIBRARYES
```
yarn add express && yarn add -D @types/express
```
### ADD TYPESSCRIPT
```
yarn add -D typescript
```
### INIT TSC
```
yarn tsc --init
```

### START PROJECT
```
yarn dev
```
### BUILD PROJECT
```
yarn build
```
